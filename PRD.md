# Product Requirements Document (PRD): "The Unicorn Founder" - 4ourmedia Game

## 1. Executive Summary
Transform the 4ourmedia website into an engaging, viral browser-based game. The game simulates the journey of a modern entrepreneur building a business from scratch using current tools (AI, SaaS, Social Media) and workflows.

## 2. Game Mechanics
### Core Loop
1.  **Click/Work:** Player clicks a central button ("Code", "Sell", "Create Content") to generate initial revenue/users.
2.  **Earn Resources:** Primary resources are **Cash**, **Users**, and **Hype**.
3.  **Invest/Upgrade:** Spend Cash to buy **Assets** (Laptops, Servers) and **Talent** (Freelancers, AI Agents, Full-time Devs).
4.  **Automate:** Hired talent and tools automate the resource generation.
5.  **Scale:** Unlock new marketing channels (TikTok, LinkedIn, SEO) to boost multipliers.

### "Viral" & Modern Features
*   **Events:** Random events based on current economy (e.g., "AI Regulation passed - productivity -10%", "Viral Tweet - Hype +500%").
*   **Tech Tree:** Unlock modern tech: No-Code Tools -> Custom SaaS -> AI Agents -> Quantum Computing.
*   **Prestige System:** "Exit/Sell Company" to restart with a multiplier (Angel Investor status).

## 3. User Interface (UI)
*   **Theme:** "Dark Mode SaaS Dashboard" aesthetic. Neon accents, clean typography (Inter/Roboto).
*   **Layout:**
    *   **Left Panel:** Stats (Cash, MRR, Valuation, Day #).
    *   **Center:** The "Grind" button and visual representation of the office (upgrades visually from garage to skyscraper).
    *   **Right Panel:** Shop/Upgrades (Tabs: Staff, Tech, Marketing).
    *   **Bottom:** Log/News Ticker (Game events).

## 4. Technical Architecture
*   **Frontend:** HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript (ES6+).
*   **State Management:** `localStorage` to save progress automatically.
*   **Performance:** Lightweight, mobile-responsive.

## 5. Implementation Plan
*   **Phase 1: Skeleton:** Set up the HTML structure for the dashboard.
*   **Phase 2: Core Logic:** Implement the game loop (ticker, resource generation, click handlers).
*   **Phase 3: Upgrades System:** Create the data structure for buying items and calculating costs.
*   **Phase 4: UI/UX Polish:** Apply the "SaaS Dashboard" CSS theme.
*   **Phase 5: Events & Content:** Add the flavor text and random events.
